## BitBucket 활용
- 깃 명령어
  * pull
    * 가져 오기 (서버 소스 저장소의 소스를 로컬에 가져온다.) 
  * add
    * 추가, 로컬 소스 저장소에 해당 파일이 생성 된 것을 알린다.
  * commit
    * 로컬 소스 저장소에 파일의 수정 사항을 기록
  * Push
    * 서버의 소스 저장소에 기록된 내용들을 전송
